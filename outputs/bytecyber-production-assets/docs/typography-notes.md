# ByteCyber Typography Notes

## Primary Sans

Use a neutral, modern sans-serif for headings, body text, navigation, buttons, and product copy.

Recommended stack:

```css
font-family: Inter, -apple-system, BlinkMacSystemFont, "Helvetica Neue", Arial, sans-serif;
```

Usage:

- Headings: 620-660 weight, tight but readable line-height.
- Body: 400-450 weight, generous line-height.
- Buttons and navigation: 600-650 weight.
- Avoid overly decorative or futuristic typefaces.

## Mono-Style Technical Font

Use a mono-style font only for technical labels, small interface tags, metrics, status text, and infrastructure details.

Recommended stack:

```css
font-family: "SF Mono", "IBM Plex Mono", ui-monospace, Menlo, Consolas, monospace;
```

Usage:

- Labels such as `LIVE BID LOGIC`, `RTB_LATENCY`, `SIGNALS`.
- Small status chips.
- Dashboard-like metrics and technical metadata.
- Do not use mono for large marketing headlines.

## Tone

Typography should feel:

- Clear
- Technical
- Confident
- Minimal
- Programmatic-native
